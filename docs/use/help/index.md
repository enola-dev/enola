## Global Help

Invoking the CLI without any arguments, or with `help` or `--help` shows the global help:

```bash $? cd ../.././..
$ ./enola
...
```

Each sub-command's help can be shown either with `enola help SUBCOMMAND` (better) or `enola SUBCOMMAND --help` (not recommended).
