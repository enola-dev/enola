This is a JS UI widget library for a "Dynamic Form" viewer/editor for any JSON Linked Data.

## Development

1. [Install pnpm](https://pnpm.io/installation)
1. `alias p 'pnpm '`
1. `p build`
1. `python3 -m http.server`
1. open <http://0.0.0.0:8000/src/demo.html>
